# 🍽️ Recipe App

A modern and responsive **Recipe Web Application** built with **React.js** and **Vite**.  
This app allows users to explore meals, filter recipes by categories, and view detailed meal information in a clean and user-friendly interface.

## 📌 About The Project

**Recipe App** is a frontend web application designed to help users discover meals through a simple and fast user experience.  
Users can browse meal categories, filter meals dynamically, and open a dedicated page for each meal to view full details.

This project was built to practice and demonstrate:

- React component-based architecture
- Client-side routing with React Router
- Data fetching and caching with React Query
- API integration using Axios
- Responsive UI design with Tailwind CSS
- UI components with Flowbite React

---

## ✨ Features

- Browse meal categories
- Filter meals by selected category
- View detailed recipe information
- Responsive design for all screen sizes
- Reusable React components
- Fast and optimized API data fetching with React Query
- Clean and modern UI using Tailwind CSS + Flowbite React
